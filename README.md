# isCss
isCss is a Node.js package, return true if input is a valid Cascading Style Sheets


## Install

```bash
npm install iscss
```

## Usage

```js
const isCss  = require('iscss')
const css = 'body { font-size: 12px; }'

isCss(css) // return TRUE
```
